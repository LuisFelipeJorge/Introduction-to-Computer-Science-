# Description
Implement a C program that reads in the standard text input a real number **a**, an operator (char) **op** and another real number **b**. Then print the result of the mathematical operation **a op b** on the screen. The **op** operator can be one of the four characters that represent the basic functions **+, -, âˆ—** and **/**, or the **%** symbol. In this case, you must print the percentage that **a** represents from **b** (Example: **8 % 50 = 16%**).

## Input example
| Input | Output             |   
|-------|--------------------|
| 2 + 3 | 5.000000 |   
| 7 * 2 | 14.000000     |   
| 8 % 50 | 16.000000     |
